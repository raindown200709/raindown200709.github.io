【MySQL】MySQL创建、删除数据表

RainDown 2020-07-01 MySQL

MySQL数据库已经创建好了，那数据表怎么创建、删除呢？

如下实例：

创建数据表

mysql> CREATE TABLE IF NOT EXISTS `raindown_tbl`(
    ->    `raindown_id` INT UNSIGNED AUTO_INCREMENT,
    ->    `raindown_title` VARCHAR(100) NOT NULL,
    ->    `raindown_author` VARCHAR(40) NOT NULL,
    ->    `submission_date` DATE,
    ->    PRIMARY KEY ( `raindown_id` )
    -> )ENGINE=InnoDB DEFAULT CHARSET=utf8;
Query OK, 0 rows affected (0.03 sec)
mysql>

删除数据表

mysql> USE raindown;
Database changed
mysql> DROP TABLE raindown_tbl;
Query OK, 0 rows affected (0.8 sec)
mysql>
